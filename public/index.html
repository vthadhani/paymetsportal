PlaceToPay / Atlantic Bank — API integration

PlaceToPay credentials (login + secretKey) must be kept server-side. Never expose them in client-side JS. Use a backend proxy to call the PlaceToPay API and return the processUrl to this frontend.

Step 1 — Create a payment session (Node.js backend)

// POST to PlaceToPay API to create a hosted checkout session const crypto = require('crypto'); function ptpAuth(login, secretKey) { const nonce = crypto.randomBytes(16).toString('base64'); const seed = new Date().toISOString(); const tranKey = crypto.createHash('sha256') .update(nonce + seed + secretKey) .digest('base64'); return { login, tranKey, nonce: Buffer.from(nonce).toString('base64'), seed }; } const payload = { auth: ptpAuth(process.env.PTP_LOGIN, process.env.PTP_SECRET), payment: { reference: 'INV-001', description: 'Invoice for services', amount: { currency: 'BZD', total: 250.00 } }, expiration: '2025-12-31T23:59:59-06:00', returnUrl: process.env.PTP_RETURN_URL, ipAddress: req.ip, userAgent: req.headers['user-agent'] }; const res = await fetch(process.env.PTP_ENDPOINT, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify(payload) }); const data = await res.json(); // data.processUrl ← this is your shareable payment link! return data.processUrl;

Step 2 — Paste processUrl as the payment link here

The processUrl returned by PlaceToPay is your shareable payment link. Paste it into the "Create link" form (select Atlantic Bank gateway first). The customer clicks it, sees Atlantic Bank's hosted checkout, and pays with their local BZD debit card.

Step 3 — Check payment status

const statusRes = await fetch( `${process.env.PTP_ENDPOINT}/${requestId}`, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ auth: ptpAuth(login, secret) }) } ); const { status } = await statusRes.json(); // status.status === 'APPROVED' | 'PENDING' | 'REJECTED'

Gateway routing logic

// Smart routing: send to correct gateway based on card type / location function routePayment(customerCountry, cardType) { if (customerCountry === 'BZ' || cardType === 'debit') { return 'atlantic_bank_placetopay'; // local BZD debit } return 'stripe'; // international credit cards }

What to get from Atlantic Bank

Contact Atlantic Bank Belize and request:
1. Your PlaceToPay merchant login and secretKey
2. The exact PlaceToPay API endpoint URL for Belize
3. Sandbox/test credentials to verify integration before going live
4. Confirm supported currencies (BZD, USD) with the bank
